# Спецификация приложения погоды «cast»

Это приложение воссоздается на основе рассказа из блокнота «Сергей Нагорный» в NotebookLM.

## Требования к функционалу
1. **Интеллектуальный поиск (Smart Search):** Строка поиска городов с функцией автодополнения (suggestions) при вводе.
2. **Быстрый доступ:** Кнопки для моментального выбора популярных городов: Москва, Санкт-Петербург, Сочи, Казань, Екатеринбург, Владивосток, Дубай.
3. **Геолокация:** Кнопка для определения текущего местоположения пользователя по GPS (встроенная в браузер геолокация) и загрузки погоды для этих координат.
4. **Подробные метеоданные:**
   * Текущая точная температура и состояние погоды.
   * Дополнительные показатели: вероятность осадков, скорость ветра, влажность, давление, время восхода и заката.
   * Почасовой прогноз погоды (на ближайшие 12-24 часа).
   * Прогноз погоды на 5 дней с минимальной/максимальной температурой.
5. **Дизайн («Атмосферный дизайн»):**
   * Премиальный стиль Glassmorphism (эффект матового стекла, `backdrop-filter: blur`).
   * Динамичное неоновое фоновое свечение (neon glow), меняющее свой цвет в зависимости от текущей погоды (солнечно, дождь, гроза, снег, облачно).
   * Красивые современные иконки для каждого типа погоды.

## Архитектура и API
* **Frontend:** Чистый HTML5, CSS3, ES6 JavaScript. Без внешних тяжелых фреймворков и библиотек.
* **Weather API:** Бесплатное API [Open-Meteo](https://open-meteo.com/) (не требует ключей API).
* **Geocoding API:** [Open-Meteo Geocoding API](https://open-meteo.com/en/docs/geocoding-api) для автодополнения городов при поиске.
